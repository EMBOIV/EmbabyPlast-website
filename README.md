# Embaby Plast - Plastic Comb Manufacturer Website

A professional B2B website for **Embaby Plast**, a plastic comb manufacturing factory. Built with HTML, CSS, and vanilla JavaScript.

## Features

- **Product Catalog** with 7 comb product types (Hair Styling, Barber Professional, Specialty)
- **Product Detail Pages** with specifications (material, size, MOQ)
- **B2B Inquiry System** — add products to inquiry list and send via WhatsApp
- **Saved Products** (wishlist) with localStorage persistence
- **Responsive Design** — mobile-friendly with industrial theme
- **Category Filtering** on the catalog page

## Pages

| Page | Description |
|------|-------------|
| index.html | Homepage with factory info, product range, export stats, contact |
| products.html | Full product catalog with category filters |
| product.html | Individual product detail with specs and inquiry button |
| cart.html | Product inquiry list with WhatsApp submission |
| wishlist.html | Saved products for later review |

## Tech Stack

- HTML5 / CSS3 / Vanilla JS
- Google Fonts (Montserrat, Inter)
- Font Awesome 6.4 icons
- localStorage for cart and wishlist
- WhatsApp Business API for inquiry submission
